# 3D scene in OpenGL
This is a university term project on OpenGl. It is written in C++ and GLSL.
The RV scene in New Mexico desert with 3D objects and sound effects.

Here are the libraries that have been used here (they are free):
- OpenGl;
- stb_image;
- glfw;
- glm;
- Assimp;
- SFML (Audio).
